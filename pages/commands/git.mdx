# Git Commands

## toga-cli git gone (or toga-cli git g)

This command removes all branches that no longer exist on the remote repository. It helps to keep your local repository clean and up-to-date by deleting branches that have been removed from the remote.

### Usage

```sh
toga-cli git gone
# or
toga-cli git g
```

### Example

```sh
# Remove all branches that do not exist on the remote anymore
toga-cli git gone
```

After running this command, any local branches that have been deleted from the remote repository will also be removed from your local repository.
